Surgimento da internet

(Voltar)

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos. Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documento, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações. Ao visitar uma página simples na web, você pode perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa estrutura é estabelecida através do HTML. No inicio da web, era comum encontrar sites apenas contendo textos e imagens simples, com estrutura básica e sem estilizações. Porém, nos dias atuais, muito dificilmente você encontrará sites que possuam apenas elementos HTML. Portanto, podemos considerar o HTML o "esqueleto" da sua página. Imagine então que além do esqueleto, é necessário ter o corpo. Para isso, temos então as linguagens CSS e o JavaScript, que em conjunto com HTML, formam a base para todos os websites atuais. Veremos mais à frente o que significam essas linguagens.

Como funciona o HTML

(Voltar)

Através de um documento HTML, ou seja, um documento com a extensão .html ou .htm., o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá- lo. Os arquivos .html podem ser visualizados em qualquer navegador (como Google Chrome, Safari, ou Mozilla Firefox). Geralmente um site é composto por diversas páginas HTML, como por exemplo: um website que contenha três páginas (uma homepage, uma página de contato e uma página de produtos) receberá ao menos três documentos .html distintos, sendo uma para cada página do website. O código pode ser escrito através de qualquer editor de texto, como o próprio bloco de notas. Cada página consiste em uma série de tags (também chamados de elementos) que podem ser considerados os blocos de construção das páginas. Portanto, esses blocos são a maneira com a qual o HTML faz a marcação dos conteúdos, criando a hierarquia e a estrutura do mesmo, dividido entre seções, parágrafos, cabeçalhos, e outros.

Exemplos de tags HTML

(Voltar)

Com qualquer editor de texto, como Sublime Text, NotePad++ ou até mesmo o Bloco de Notas, é possível criar um documento com a extensão .html que pode ser exibido em navegadores.

Este documento HTML é composto por uma série de tags, que são códigos responsáveis por definir toda a estrutura da página, incluindo elementos como o título, parágrafos, quebras de linha, e outros. A maioria dos elementos em um documento HTML segue uma estrutura de abertura e fechamento de tags, como <h1> e </h1>. Além disso, há tags que servem para inserir quebras de linha, como <br>.

Por exemplo, se você quiser escrever um parágrafo, usará a tag <p>. O parágrafo é escrito entre a tag de abertura <p> e a tag de fechamento </p>. Veja o exemplo:

<p>Meu primeiro parágrafo.</p>

Ao salvar o arquivo com a extensão .html e abri-lo em um navegador, você verá o parágrafo exibido na tela.

Outras Tags HTML Importantes

Além da tag <p>, existem várias outras tags HTML importantes que você pode utilizar para estruturar e formatar seu conteúdo. Aqui estão algumas delas:

  • <h1> até <h6> – Definem diferentes níveis de cabeçalhos, sendo <h1> o mais importante e <h6> o menos importante.
  • <mark> – Destaca texto com um fundo amarelo.
  • <small> – Reduz o tamanho do texto.
  • <i> – Torna o texto itálico.
  • <u> – Sublinha o texto.
  • <strong> – Dá ênfase ao texto, geralmente em negrito.
  • <ol> – Cria listas ordenadas.
  • <ul> – Cria listas não ordenadas.
  • <li> – Define um item de lista.
  • <a> – Define um link.
  • <hr> – Insere uma linha horizontal.
  • <sub> – Define texto em subscrito.
  • <sup> – Define texto em sobrescrito.
  • <blockquote> – Define uma citação em bloco.

Novas Tags Sugeridas

Além das tags mencionadas, você pode utilizar novas tags sugeridas para maior flexibilidade:

  • <font> – Define a fonte, tamanho e cor do texto. Embora obsoleta, é útil para compatibilidade com documentos antigos.
  • <del> – Marca texto que foi removido.
  • <abbr> – Define uma abreviação ou acrônimo.
